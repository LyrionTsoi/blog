# [AcWing 4714.数对](https://www.acwing.com/problem/content/4717/)
给定一个长度为 N 的字符串 S，字符串中的字符下标从左到右依次为 1∼N。

请你计算共有多少个数对 (i,j) 能够同时满足以下条件：

1≤i,j≤N（i 和 j 可以相等）
S[i]=S[j]
注意，(2,1) 和 (1,2) 视为两个不同的数对。

输入格式
共一行，一个字符串 S。

输出格式
一个整数，表示满足条件的数对数量。

数据范围
前三个测试点满足 1≤N≤10。
所有测试点满足 1≤N≤105，字符串中只包含小写字母和数字。

输入样例1：
great10
输出样例1：
7
输入样例2：
aaaaaaaaaa
输出样例2：
100
